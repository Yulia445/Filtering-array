# Filtering-array

## Опис
У цій програмі я створив функцію filtrovane, яка приймає масив і розділяє всі елементи за типами. Спочатку я перевіряю, чи це взагалі масив. Потім проходжуся циклом по всіх елементах і розкладаю їх у різні списки: числа, строки, null та undefined. Числами я також вважаю ті строки, які можна перетворити на число. В кінці функція повертає масив із чотирма категоріями, щоб було зручно бачити, що куди потрапило.
